Mouseless Menus

A tool with a mouse-free environment, suitable for touch-typists and those who prefer to use keyboards. Saves your wrists from doing the mouse-shuffle :-)

IBM iSeries Menu System - a mouse-free alternative

Based on the foundations of the IBM mid-range idea. A menu of options can be used more easily by picking a number that's shown before a menu item description.

This app uses similar function keys as the IBM mid-range systems. These include:

  1. F1 - Help
  2. F3 - quit
  3. F5 - refresh
  4. F7 - edit the current menu
  5. F8 - copy text selection from the audit log below the commands
  6. F9 - recall previous command
  7. F10 and F12 - backstep to previous menu
  8. F13 - View 'To Do' list - not an IBM feature !
  9. F15 - See all available menus
  10. F17 - a toggle to view the command behind each menu entry

More custom menus can be built by any notepad or edit program that does not decorate the text with styling such as bold, etc.

Function key one is the menu help key and will bring up a web page with more details than we can describe here.
