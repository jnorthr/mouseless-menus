Mouseless Menus

A tool with a mouse-free environment that's suitable for touch-typists and those who prefer to use keyboards. Saves on your wrists from the mouse-shuffle :-)

IBM iSeries Menu System - a mouse-free alternative

Based on the foundations of the IBM mid-range idea. A menu of options can be picked more easily by using a number that's shwon before the menu item description.

This app uses similar function key combinations as seen on IBM mid-range systems. More custom menus can be built by any notepad or edit program that does not decorate the text with styling such as bold, etc.

Function key one is the menu help key and will bring up a web page with more details than we can describe here.
